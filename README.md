Руководство по настройке слайдера:

Все управление параметрами слайдера производится через изменение значений у полей dataset в документе разметки index.html.
Основные параметры:

1. data-slides-to-show="number int" - задает количество слайдов для одновременного отображения в контейнере слайдера. [мин: 1, макс: количество слайдов];
2. data-slides-to-scroll="number int" - задает количество слайдов для скроллинга (вручную через кнопки управления и автоматического скроллинга) [мин: 1, макс: data-slides-to-show];
3. data-start-slide="number int" - задает индекс первого слайда для отображения [мин: 1, макс: количество слайдов];
4. data-autoscroll-delay="number int" - задает промежуток времени в секундах для автоматического скроллинга слайдера [мин: 0, макс: n]

Дополнительные параметры:

1. data-controll-button-align="text row " - задает положение кнопок скроллинга слайдера [inside - кнопки находятся внутри контейнера слайдера, outside - кнопки находятся снаружи контейнера слайдера, none - отключение кнопок скроллинга]
2. data-indicator-align="text row" - задает положение индикатора слайдера [top inside - индикатор сверху внутри слайдера, top outside - индикатор сверху снаружи слайдера, bottom inside - индикатор снизу внутри слайдера, bottom outside - индикатор снизу снаружи слайдера]
3. data-animation-scroll="text row" - задает тип анимиции скроллинга слайдера [smooth - плавный переход, instant - мгновенный переход, auto - переход установленный вашим браузером по-умолчанию]
